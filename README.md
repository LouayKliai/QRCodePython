QR Code Generator in Python

This Python script allows you to generate a QR code from a link or text that you provide as input.

Prerequisites:
Python 3.x installed on your system

Usage:
Run the script using Python.
When prompted, enter the link or text you want to encode into the QR code.
The script will generate a QRCODE.png file containing the corresponding QR code.
You will see the message "QR Code generated successfully!" once the QR code file has been created successfully.

#How to Run the Script:
python main.py
